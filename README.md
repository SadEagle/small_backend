# Описание

Данный проект представляет из себя пример бэкенда с использованием FastAPI.

Реализованные составляющие проекта:
- [x] FastAPI
- [x] SQLAlchemy ORM + Pydantic
- [x] Pytest
- [ ] Alembic migrations
- [ ] Logs
- [ ] Docker/Compose

# Запуск программы

Поднятие контейнера:
```
docker compose up
```

Запуск тестов:
```
uv run pytest
```
